## [0.3.0] – Safety & Intelligence Layer
### Added
- Safety Readiness system with ethical, non-alarming language
- Safety Legend explaining score interpretation
- “Why this safety score?” expandable per heritage site
- Per-factor safety breakdown (lighting, roads, emergency access, network, feedback)
- Risk-based heatmap overlay (AI-ready)

### Improved
- Transparency and trust in heritage exploration
- User confidence through explainable safety indicators

---

## [0.2.0] – Map Intelligence & Discovery
### Added
- Interactive Leaflet-based map for Chambal region
- Marker clustering for scalable heritage discovery
- Custom heritage-style cluster badges
- Heatmap toggle for density, risk, and popularity
- Smooth fly-to animation on selection

### Improved
- Map performance at scale
- Visual storytelling and exploration UX

---

## [0.1.0] – Core MVP (Hackathon Release)
### Added
- Next.js (App Router) frontend with TypeScript & Tailwind
- Chambal region landing page with historical context
- Heritage site cards with map synchronization
- Card ↔ marker interaction (two-way sync)
- Initial data model for heritage sites

### Technical
- Client-side architecture safe for SSR
- Modular, scalable component structure
